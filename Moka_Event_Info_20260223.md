# 真岡市周辺イベント情報 (2026年2月23日)

2026年2月23日（月・祝）に栃木県真岡市およびその周辺で開催される主なイベント情報は以下の通りです。

## 1. 真岡・浪漫ひな飾り
- **開催期間:** 2026年2月1日 〜 3月3日
- **場所:** 久保講堂（メイン会場）および真岡市街地
- **内容:** 
  - 豪華絢爛な26段（約1,000体）のひな飾り。
  - 伝統的な吊るし雛の展示。
  - 真岡の歴史的な建物を舞台にした風情ある雛祭りを楽しめます。

## 2. マルフクマーケット (FKDインターパーク店)
- **開催期間:** 2026年2月21日 〜 2月23日
- **場所:** FKDインターパーク店（真岡市隣接の宇都宮市上三川町）
- **内容:** 
  - 真岡市内のいちご農園が出店。
  - JGAP認証を受けた特別なイチゴや、イチゴを使ったスイーツ、こだわりのフード、雑貨など約20店舗が集結します。
  - 三連休の最終日として、家族連れで賑わうことが予想されます。

## 3. 真岡鐵道 SLもおか号
- **注意点:** 2026年度は運休となる予定です。鉄道ファンや観光で検討されている場合はご注意ください。
